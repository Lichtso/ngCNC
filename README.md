# ngCNC
Plans for an Successor of the PrismCNC

## Machining Hardware
- Acrylic Box
  - With Doors / Hatches
- 3 Axis Linear
- 2 Axis Angular
- Stepper Motor Drivers
- Position Measurement
- Tool Length Measurement
- Automatic Tool Changer
- Spindle Motor & Driver
  - Speed Control
  - Power Control

## Power & Control Hardware
- Metal Box
- Plugs & Cables
- Emergency Shutdown Button
- Arduino for Realtime Control
- Raspery PI or similar
  - Ethernet Port
- Power Supply
  - Transformer
  - On / Off Switch
  - Power Socket (IEC 60320, C13)

## Software
- Movement Control
- Web Interface
  - Remote Control
  - Upload G-Code
